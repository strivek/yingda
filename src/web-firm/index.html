关于我们

应大创意产业园全景

天津应大股份有限公司成立于1992年，以经营“应大”品牌为主，是一家集品牌经营、研发设计、生产销售为一体的专业化、现代化的皮衣皮草公司，拥有国内首家皮衣皮草博物馆和国内最大的皮衣皮草产业园。

公司自成立之日起，秉承“百年应大、诚信应大、创新应大、激情应大”的企业精神，连续多年皮革市场综合占有率居全国第一；先后荣获“中国名牌”、“中国驰名商标”和“国家免检产品”荣誉称号；连续六次获得“中国真皮衣王”；连续八次代表中国皮装行业发布流行趋势，成为行业风向标。应大的快速发展也得到了党和政府的肯定。2005年10月1日，胡锦涛总书记在接见应泽从董事长时，寄语应大“应大应大，应该发展壮大”。我们将锐意进取，将应大打造成世界上最好的皮衣皮草品牌。
