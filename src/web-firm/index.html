关于我们

应大公司始创于1992年，经过多年潜心经营，于2005年推出具有国际水准的百万巨制——"中华衣王"，在中国时尚界引起极大轰动；同年，胡锦涛总书记考察天津时来到应大，寄予了"应大应大，应该发展壮大"的厚望。此后，应大开始建立皮衣工业园，并多次荣登“中国500最具价值品牌排行榜”。

在卓力打造皮衣服饰文化的同时，应大始终坚持"百年应大，诚信应大，创新应大，激情应大"的企业精神，向着"创建中华民族的世界品牌"的宏伟目标迈进。
